### Unblock the File to bypass the strict execution policy

Windows flags files downloaded from the internet. To remove this flag:

- Right-click the .ps1 file in File Explorer and select Properties.

- At the bottom of the General tab, look for a Security section.

- Check the "Unblock" box and click Apply.
